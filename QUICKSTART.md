# 🚀 Quick Start Guide

## What You Have

This ZIP contains a complete WhatsApp marketing automation system:

- ✅ 2 n8n workflows (ready to import)
- ✅ 4 Google Sheets templates (ready to import)
- ✅ 30+ WhatsApp message templates (ready to use)
- ✅ Helper scripts & utilities
- ✅ Docker deployment config
- ✅ Complete documentation

## Steps to Get Started

### 1. Extract This ZIP

```bash
unzip whatsapp-marketing-automation.zip
cd whatsapp-marketing-automation
```

### 2. Read Documentation

Start here: `docs/SETUP_GUIDE.md`

### 3. Setup Services

- Google Sheets (follow docs/SETUP_GUIDE.md)
- WhatsApp API (Twilio/360Dialog/Meta)
- n8n (npm install n8n -g)

### 4. Import & Configure

- Import Google Sheets CSVs
- Import n8n workflow JSONs
- Edit config/.env with credentials

### 5. Test & Deploy

- Send test messages
- Verify replies work
- Check email alerts
- Go live!

## Need Help?

- Read docs/ folder thoroughly
- Check README.md in each directory
- Review example templates in whatsapp-templates/
