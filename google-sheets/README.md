# Google Sheets Templates

## Import Instructions

1. Create a new Google Sheet
2. Create 4 tabs: LEADS_MASTER, MESSAGE_CALENDAR, MESSAGE_LOG, REPLY_LOG
3. Import CSV files from `sheet-templates/` into respective tabs
4. See [Setup Guide](../docs/SETUP_GUIDE.md) for Google Sheets API setup

## Sample Data

Sample data is provided in `sample-data/` folder for testing.
