# Configuration Guide

## Quick Setup

1. Copy `.env.example` to `.env`
2. Fill in your credentials
3. See [Setup Guide](../docs/SETUP_GUIDE.md) for details

## Providers Supported

- Google Sheets API
- Twilio WhatsApp API
- 360Dialog WhatsApp API
- Meta WhatsApp API
- Gmail SMTP

For detailed setup instructions, see the main documentation.
