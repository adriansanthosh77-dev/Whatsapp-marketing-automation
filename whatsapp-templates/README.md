# WhatsApp Message Templates

## Template Creation

1. Write templates in WhatsApp Business Manager
2. Submit for approval (24-48 hours)
3. Get Template ID
4. Add to MESSAGE_CALENDAR in Google Sheets

## Template Format

See .txt files in subdirectories for examples.

Variables use {{1}}, {{2}} format.

For detailed guide, see [Template Guide](../docs/TEMPLATE_GUIDE.md).
